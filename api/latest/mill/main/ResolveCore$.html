ResolveCore

object ResolveCore

Takes a single list of segments, without braces but including wildcards, and resolves all possible modules, targets or commands that the segments could resolve to.

Returns a Result, either containing a Success containing the Resolved set, NotFound if it couldn't find anything with some metadata about what it was looking for, or Error if something blew up.
